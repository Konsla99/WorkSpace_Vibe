# Gemini Career Assistant

이 애플리케이션은 Gemini CLI를 활용하여 사용자의 취업 준비 과정을 돕는 데 특화된 GUI 도구입니다. 복잡한 취업 준비 과정을 효율적으로 지원하고, Gemini AI의 강력한 기능을 통해 사용자 경험을 향상시키는 것을 목표로 합니다.

## ✨ 주요 기능

1.  **직무 분석**: 입력된 채용 공고를 기반으로 필요한 핵심 역량과 직무 관련 정보를 분석합니다.
2.  **자소서 초안 작성**: 분석된 직무 정보와 사용자 데이터를 활용하여 자기소개서 초안을 자동으로 생성합니다.
3.  **첨삭 피드백**: 작성된 자기소개서에 대해 Gemini AI가 구체적인 개선점과 피드백을 제공합니다.
4.  **면접 질문 도출**: 자기소개서 및 예상 직무를 바탕으로 면접에서 나올 수 있는 질문들을 예측하고 준비를 돕습니다.
5.  **AI 표현 다듬기**: 사용자의 원문을 받아 Gemini AI를 통해 더 전문적이고 효과적인 표현으로 다듬어줍니다.

## ⚠️ 필수 요구 사항

본 프로그램을 정상적으로 사용하기 위해서는 반드시 시스템에 `gemini-cli`가 설치되어 있어야 합니다.

### Gemini CLI 설치 방법
터미널(PowerShell 또는 CMD)에서 아래 명령어를 입력하여 설치하세요:
```bash
npm install -g @google/gemini-cli
```
설치 후 `gemini` 명령어가 터미널에서 정상적으로 작동하는지 확인해야 합니다.
