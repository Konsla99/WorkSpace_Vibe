# Gemini Career Assistant 설정 방법

이 프로그램은 **Gemini CLI**를 기반으로 작동하는 GUI 도구입니다. 정상적인 사용을 위해 아래 설정을 완료해 주세요.

## 1. Node.js 설치
* **공식 사이트:** [https://nodejs.org/](https://nodejs.org/)
* **LTS 버전**을 다운로드하여 설치해 주세요.

## 2. Gemini CLI 인증 설정
터미널(PowerShell 또는 CMD)을 열고 아래 명령어를 입력하여 인증을 완료하세요.

### 구글 로그인 인증 (권장)
```powershell
gemini --auth
```
* 위 명령어를 입력하면 브라우저가 열립니다. 사용 중인 구글 계정으로 로그인하여 인증을 완료해 주세요.

### API 키 설정 (필요한 경우)
만약 로그인 인증이 어렵다면 API 키를 직접 설정할 수도 있습니다.
1. [Google AI Studio](https://aistudio.google.com/app/apikey)에서 API 키를 발급받습니다.
2. 터미널에 아래 명령어를 입력합니다.
```powershell
setx GEMINI_API_KEY "여러분의_API_KEY"
```
* **주의:** 설정 후 터미널이나 프로그램을 재시작해야 적용됩니다.

## 3. 프로그램 사용법
1. **워크스페이스 지정:** 작업할 파일들이 들어있는 폴더를 선택합니다.
2. **업무 시작:** 버튼을 누르면 해당 폴더에 `지침.md` 파일이 생성됩니다.
3. **App 설정 알리기:** Gemini가 `지침.md`의 규칙을 인식하도록 합니다.
4. **기능 실행:** 왼쪽 메뉴의 1\~5번 버튼을 눌러 작업을 진행합니다.
