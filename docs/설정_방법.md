# Resume_Gem 사용 매뉴얼

`Resume_Gem`은 Gemini AI를 활용하여 직무 분석부터 자소서 첨삭까지 도와주는 커리어 어시스턴트입니다. 아래의 순서에 따라 사용해 주세요.

## 🚀 사용 시퀀스 (Sequence)

### 1단계: 워크스페이스 지정 (필수)
*   **방법**: 좌측 상단의 **[📁 워크스페이스 지정]** 버튼을 클릭하여 작업할 폴더를 선택합니다.
*   **주의**: 워크스페이스를 지정하지 않으면 Gemini 엔진이 실행되지 않으며, 모든 기능 버튼이 비활성화 상태로 유지됩니다.

### 2단계: Gemini 로딩 대기
*   **현상**: 워크스페이스 지정 후 터미널에 메시지가 출력되며 Gemini가 로딩됩니다.
*   **확인**: 상단 **System Status**가 파란색 '호출 중...'에서 초록색 **'Gemini 준비 완료'**로 바뀔 때까지 기다려 주세요.

### 3단계: 희망 직무 입력 및 페르소나 설정
*   **방법**: **🎯 희망 직무** 칸에 분석을 원하는 직무(예: 임베디드 개발, 마케팅)를 입력하고 **[App 설정 알리기]** 버튼을 누릅니다.
*   **주의**: 직무를 입력하지 않으면 알림이 뜨며 진행되지 않습니다. 이 버튼을 눌러야 AI가 전문가 모드로 전환되며 하단의 기능 버튼들이 활성화됩니다.

### 4단계: 파일 선택 및 기능 실행
*   **파일 선택**: 좌측의 **File Selection** 그룹에서 필요한 파일(JD, 자소서 등)을 먼저 선택합니다.
*   **실행**: 하단 바에서 원하는 기능을 클릭합니다. (예: 직무 분석, 자소서 초안 등)
*   **결과 저장**: 답변이 만족스럽다면 좌측의 **[💾 결과 md 파일로 저장]** 버튼을 눌러 파일로 보관할 수 있습니다.

---

## ⚠️ 주의 사항 및 팁

1.  **엔터 누락**: 간혹 시스템 부하로 인해 명령어가 전송된 후 엔터가 입력되지 않을 수 있습니다. 이 경우 터미널을 한 번 클릭하고 키보드의 엔터를 직접 눌러주세요.
2.  **직무 수정**: 직무를 잘못 입력했다면 입력창 옆의 **[수정]** 버튼을 눌러 페르소나를 다시 설정할 수 있습니다.
3.  **파일 경로**: 파일명이나 경로에 특수문자가 너무 많으면 인식이 어려울 수 있으니 가급적 깔끔한 파일명을 권장합니다.
4.  **창 크기**: 터미널 글자가 깨져 보인다면 창 크기를 살짝 조절해 보세요. 자동으로 레이아웃이 재계산됩니다.
